#  Air Quality Index Dashboard (Power BI)

This project presents an interactive **Air Quality Index (AQI) Dashboard** built using Microsoft Power BI to analyze pollution levels across cities and over time.

---

##  Objective

The goal of this dashboard is to visualize air pollution trends and identify the most polluted cities using AQI and pollutant data.
It helps users quickly understand environmental conditions and patterns through visual insights.
